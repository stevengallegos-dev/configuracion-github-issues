# Historia 4: Agregar Productos al Carrito

## Historia de Usuario

Como cliente de LePetitDep

Quiero agregar productos al carrito de compras

Para preparar mi pedido antes de confirmar la compra.

## Criterios de Aceptación

- El usuario puede seleccionar uno o varios productos.
- El usuario puede elegir la cantidad deseada de cada producto.
- El carrito actualiza automáticamente el total de la compra.
- El usuario puede eliminar productos del carrito.
- Los productos permanecen guardados en el carrito hasta finalizar el pedido.
