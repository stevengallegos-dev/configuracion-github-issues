# Historia 2: Inicio de Sesión

## Historia de Usuario

Como usuario registrado

Quiero iniciar sesión con mi correo electrónico y contraseña

Para acceder a mi cuenta y realizar pedidos.

## Criterios de Aceptación

- El usuario puede ingresar correo electrónico y contraseña.
- El sistema valida las credenciales ingresadas.
- Si las credenciales son correctas, permite el acceso.
- Si las credenciales son incorrectas, muestra un mensaje de error.
- La sesión permanece activa hasta que el usuario cierre sesión.
