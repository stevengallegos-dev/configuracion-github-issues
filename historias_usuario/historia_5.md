# Historia 5: Realizar Pedido

## Historia de Usuario

Como cliente de LePetitDep

Quiero confirmar mi pedido

Para recogerlo posteriormente en el local.

## Criterios de Aceptación

- El sistema muestra un resumen del pedido.
- El usuario puede seleccionar un método de pago simulado.
- El sistema genera un número de orden único.
- El sistema muestra una confirmación del pedido realizado.
- El pedido queda registrado en el historial de pedidos.
