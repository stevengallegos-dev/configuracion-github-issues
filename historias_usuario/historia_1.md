# Historia 1: Registro de Usuario

## Historia de Usuario

Como cliente de LePetitDep

Quiero crear una cuenta utilizando mi correo electrónico y una contraseña

Para poder acceder a la aplicación y realizar pedidos.

## Criterios de Aceptación

- El usuario puede ingresar correo electrónico y contraseña.
- El correo debe tener un formato válido.
- La contraseña debe tener al menos 8 caracteres.
- El sistema no permite correos ya registrados.
- El sistema muestra un mensaje de registro exitoso.
